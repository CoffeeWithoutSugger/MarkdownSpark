# MarkdownSpark
这是一个简单的markdown实时预览编辑器

功能如下:
> - 欢迎界面
>   - 查看已保存的标签名称以及保存时间 
> - 编辑页面
>   -  输入 / 进行提示快捷功能提示,上下键进行选择,Enter或点击进行确认
>   -  可保存标签(内容保存至浏览器缓存中),在欢迎界面进行展示
>   -  可对保存的标签进行二次编辑 可覆盖保存或者保存新标签
